Emments

emments are shortcuts abbrevention for writing the html and css code.

Shortcuts Description
! it is Expands to the complete html5template
tagname Expand the complete the tag
> Used to create child element within the parent element
+ Used to create sibling elements
^ Moves up one level in the hierachy,allowing you to add sibling elements to parents elements
* Create multiple elements
text{} we can use curly braces to add text to elements
